# DevPlay Accounts Store

مشروع مستقل لعرض حسابات الألعاب للعملاء وربطه بنفس Supabase الخاص بلوحة الأدمن.

## التشغيل
1. افتحي `js/supabase.js`.
2. ضعي `SUPABASE_URL` و `SUPABASE_ANON_KEY`.
3. شغلي SQL الموجود في `supabase/policies.sql`.
4. ارفعي الملفات كما هي على GitHub.
5. من GitHub Pages اختاري: Deploy from a branch → main → / root.

## ملاحظة مهمة
لو الريبو اسمه `acounts` أو أي اسم، الملفات دي لا تعتمد على اسم الريبو؛ كل الروابط نسبية وتشتغل على GitHub Pages أو دومين خاص.
